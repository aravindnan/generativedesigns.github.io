Bubble Sort

Description

Visual representation of the bubble sort algorithm to sort an array of elements and represent in the form of an animation

Logic Used

A random array of elements is created displayed in the form of rectangles. The the rectangles are moved based the height which corresponds to its value in the array

Functions Used

bsort( ) - to implement the bubble sort algorithm in the array

printsort( ) - to print the semi-sorted array at each frame

makearray( ) - to create a random array of elements
