Generative Designs Using Programming

Generative Designs

Aesthetics or the sense of art has always been and always will be a way of life of humans. The expression of creativity and emotions through pictures and videos is an industry of its own. Technology ,let alone computer science has paved a sustainable way to create art with softwares providing to create Animations, Visual Effects(VFX),Computer Generated Imagery(CGI) and many more

In this project we are exploring how we can implement programming concepts to create artworks , illustrations and animations inspired from real world images, user input and computer generated randomness. This technology is the fundamental concept for computer generated data visualization which is widely used in many areas of research.

In this project we are tring to create something from the domain of computer graphics which can be implemented in the web browser with the help of HTML and a few JavaScript libraries mentioned above.

Processing is a foundation which helps artists and engineers to provide functionalities to express their creativity from scratch using its programming environment. p5js is a javascript library which is built in accordance with processing to make creative coding available in the browser.

Tools and Libraries

Procesing

Processing is a free graphical library and integrated development environment built for the electronic arts, new media art, and visual design communities with the purpose of teaching non-programmers the fundamentals of computer programming in a visual context.

p5js

p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! p5.js is free and open-source because we believe software, and the tools to learn it,should be accessible

TensorFlow

TensorFlow is an end-to-end open source platform for machine learning. It has a comprehensive, flexible ecosystem of tools, libraries and community resources that lets researchers push the state-of-the-art in ML and developers easily build.

ml5js

ml5.js aims to make machine learning approachable for a broad audience of artists, creative coders, and students. It consists of pre-trained models for detecting human poses, generating text, styling an image with another, composing music, pitch detection and many more
