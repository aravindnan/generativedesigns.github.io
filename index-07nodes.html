Nodes in free Space

Description

An Animation describing the movement of nodes moving in a confined space randomly

Logic Used

Each node position is incremented in each frame to achieve the animation and the distance between the nodes are mapped to a quantity to control the linewidth between the nodes

Functions Used

update( ) - to periodically change the position of each node

getDistance( ) - to calculate the distance between each node

draw( ) - to display each node in each frame
