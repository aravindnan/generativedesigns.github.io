Face Detection

Description

A simple implementation of face tracking model to understand the position of facial elements in a live feed using CLM Tracker

Logic Used

Coordinates of facial elements are mapped in to an array and lines are draw from 4 locations in the canvas

Functions Used

tracker( ) - clm library function to create an object of clm

getCurrentPosition( ) - to obtain the coordinates of the face

line( ) - to draw lines between 2 points
