Binary Search

Description

Visual representation of the binary search algorithm in arrays. Based on user input search value.

First Set a value -> search -> reset

Logic Used

A random array is created with elements in it . User input is taken for the search value and search is done by the binary search algorithm and result is shown

Functions Used

bbox( ) - to display each element of the array with different dimension,color and position

bsearch( ) - to perform the binary search algorithm and call bbox based on position and color
