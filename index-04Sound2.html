Oscillator

Description

This is a Frequency/Amplitude oscillator which produces sound based on a particular value of frequency at a particular amplitude and visually represent it by ellipse in free space.

Logic Used

Based on the mouse input by the user the coordinates are mapped with the frequency and amplitude values. Which would be the minimum at the left bottom corner and maximum athe right top corner of the canvas

Functions Used

freq( ) - sets the frequency of oscillator to a particular value

amp( ) - sets the amplitude of oscillator to a particular value

circles( ) - user defined function to draw circles
