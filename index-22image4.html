Video Processing

Description

This program takes live video feed from the webcam and intreprets the color values of each pixel and shows the video back in the form of hexagons

Logic Used

Each pixel of the live feed is mapped into a polygon and reprinted at each frame randomly and the resolution or the size if the polygon is controlled by the slider

Functions Used

createCapture( ) - to take video input from webcam

get( ) - to obtain color values of pixels in the video

Move the Slider to change Pixalation
