2D Noise

Description

This is an animation using Perlin Noise. A type of gradient noise used by visual effects artists to increase the appearance of realism in computer graphics.

Logic Used

Noise values are taken and mapped into range of the angle of rotation of each rectangle and the slider value to control the diamesion of the grid

Functions Used

strokeWeight( ) - to change the size of the rectanglebased on the noise value and the slider value

noise( ) - to obtain noise value of a coordinate in a 3D plain

Move the Slider to change the number of boxes
