Hand Position

Description

The implementation of handpose library in ml5js and to create moving ellipses

Logic Used

The handpose coordinate value are mapped into an array, the array values are mapped and ellipses are drawn in each frame. Each finger is mapped to a particular ellipse hence change in the movement of a finger can be displayed

Functions Used

handpose( ) - ml5 library to access hand gesture positions in realtime

ellipse( ) - to draw circles based on the handposition values
