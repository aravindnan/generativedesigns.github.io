Amplitude Reader

Description

Speak something into the mic and see the visual change in the polygon based on the amplitude of your voice input. Provide a consistent whistling tune to see the best result

Logic Used

This program takes the input from the user in the form of realtime audio and maps the amplitude value to to range of the radius of the polygon and the size of the polygon

Functions Used

getLevel( ) - Library function to obtain the amplitude of user input

poly( ) - To draw a polygon of given radius and number of sides
