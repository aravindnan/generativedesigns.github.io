Fourier Transformation

Description

Fast Fourier Transformation (FFT) is a concept used to analyze individual audio frequencies in a sound data and isolates them within a waveform and it is pictorially represented based on a sound bars and a waveform.

Logic Used

Frequency values are mapped in to the range of the height and position of the rectangles in the sound bars. Diffrent frequrrency values are also mapped to a specific vector lect to draw the waveform.

Functions Used

waveform( ) - computes amplitude values along the time domain.

analyze( ) - amplitude values along the frequency domain.Frequencies from the low to the high
