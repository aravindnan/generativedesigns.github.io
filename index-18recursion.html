Recursion

Description

A small graphical representation of what recursion, a programming concept would look like in application

Logic Used

Each branch is draw using the same function but with different arguments for the location and length. The function is calling itself until the length is 0

Functions Used

translate( ) - to move the position of the origin to the next branch

branchup( ) - the recursive function that is being called on each frame
