Weather API

Description

An Application Programming Interface . Which allows 2 remote systems to share data or any form of information and use in one's program. Here a Weather tracking API is used to get current weather details about any city in the world

Logic Used

The API data is recieved in the format of a JSON file. Which is called by an API Request based on the user's choice of the city. the JSON details are stored into a javascript object and accessed according to need .SVG images are used to represent the symbols

Functions Used

loadJSON( ) - to make an API request call to the server hosting the weather data

select( ) - to access the DOM element in the HTML page directly.
