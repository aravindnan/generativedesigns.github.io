Object Detection

Description

To recognize everyday objects by using COCOssd which is a opensource machine learning model in Tensorflowjs

Logic Used

The model is loaded and detects the image data from the webcam and uses a pre-trained neural network to come up with an output about what the object is

Functions Used

detect( ) - user defined function to draw a polygon of N sides

getImageData( ) - to obtain the image data at a particular location of particular width and height
