Gaussian Random

Description

Representation of a Gaussian process. It is a stochastic process (a collection of random variables indexed by time or space), such that every finite collection of those random variables has a multivariate normal distribution,

Logic Used

Values of a gaussian radom functions are mapped to an array a based on that value vectors are draw from the canter of the canvas.

Functions Used

randomGaussian( ) - to calculate the gaussian random values
