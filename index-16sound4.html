Synthesizer

Description

Sound synthesizer mapped to a keypad of numbers to produce sounds of 9 different musical notes and visually represent it on a grid of numbers

Logic Used

Each tile on the grid is draw with a function with particular flag on being pressed or not and based on the number pressed a switch case assigning to a particular note to be played

Functions Used

addEventListener( ) - to obtain key input from the user

MonoSynth( ) - library function to produce musical notes based on key input using a switch case
