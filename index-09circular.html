Circular Queue

Description

Simple static representation of a circualr queue data structure with functions to insert and deleted elements in the queue based on user input

Logic Used

On each input by the user new frame is print based on the number of elements in the queue ie +1 if it input is Enqueue and -1 if the input is Dequeue

Functions Used

action( ) - to take user input and perform enqueue or dequeue based on the input

drawCircle( ) - to draw elements around a circlucar path

Press 'A' to Enqueue(insert)

Press 'D' to Dequeue(remove)
