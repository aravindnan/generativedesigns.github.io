Stack

Description

An array stack data structure is visually represented here using 3D cuboids as stack elements. With push and pop functionalities based on user input

Logic Used

Each time the user inputs ,the function checks the input and based on it the push or pop is performed and at each frame the stack is reprinted based on the number of elements in stack

Functions Used

action( ) - user defined function to push and pop elemets in stack internally based on key input

box( ) - library function to display a cuboid in a particular locaiton

Press 'A' to Push element into Stack

Press 'D' to Pop element out of Stack
