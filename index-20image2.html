Raster Image

Description

This is a raster image representation of a banana. The detail of the raster can be controlled by the slider. The animation is expected to be slower with increase in detail

Logic Used

The slider controls the value of the number of blocks and the size of the blocks which represent the image. Each block is given color and depth based on the color and brightness in the picture respectively

Functions Used

rotateY( ) - library function to rotate object around the Y axis

map( ) - to map values of a particular variable from one interval to another

Move the Slider to change Raster detail
