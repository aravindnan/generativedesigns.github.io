Grid Pattern

Description

A 6 X 6 changing square grid with each tile having a random probability of being highlighted.

Logic Used

Each tile is seperately being print on each frame with a 0.4 chance of being highlighted or internally a tile being printed at the same center with a reduced radius

Functions Used

random( ) - library function to obtain a random between 0-1

rect( ) - library function to draw a rectangle at given center with given diamensions
