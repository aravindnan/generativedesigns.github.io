Infinite Pattern

Description

Floral patterns being generated which are non repeating. randominzation is done based on the number of sides of the polygon, radius of the shapes and colors used.

Logic Used

Each frame is used to print different pattern combination with a frame rate of 1. Each group of elements are drawn using different functions with appropriate arguments

Functions Used

Poly( ) - user defined function to draw a polygon of N sides

pointOnCircle( ) - to calculate the coordiantes a round a given circle
