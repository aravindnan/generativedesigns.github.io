3D Spiral

Description

3-diamentional cuboids moving around a circular spiral path with its own rotation speed and direction along 3 different axises

Logic Used

X and Y coordinates of the cuboids are found using the sine and cosine functions with a particular radius and a particular angle of rotation

Functions Used

anim( ) - user defined function to draws a 3D object and rotates it based on condition at a particular coordinate

pointLight( ) - to set the lighting for the 3D object from a particular point
