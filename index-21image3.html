Edge Detection

Description

Manually finding edges in an image using RGB values of pixels in an image

Logic Used

The average of the RGB values of each pixel is reduced by the pixel on its right side. That is if the color is same,there is no edge but if there is a change in the value it would create an edge

Functions Used

image( ) - t0 display an image in the canvas at a given location

updatePixels( ) - to change the pixel values to a new value or update to a new value
