Image Resolution

Description

Taking color values of pixels from a given image and printing squares with that same color to replicate the effect of pixalation

Logic Used

Each pixel of the image is traversed by a loop and based on the slider value the increment in the loop is changed that is that many pixels are skipped in the iteration

Functions Used

imageReso( ) - To draw rectangles on the canvas by taking color values from the image

loadImage( ) - to load an image into the system memory

Move the Slider below the image to change resolution
